Esta es una versión preliminar del TFG de José Joaquín Magaña. Para garantizar el funcionamiento, un usuario cuando confirma por correo electrónico automáticamente se le otorga el acceso 'administrador'.

Dicha funcionalidad no estará presente en la versión definitiva, por lo que cuando confirme, tendrá el acceso 'usuario', solo pudiendo modificarlo otro usuario con acceso 'administrador'.

Tampoco está disponible la modificación y eliminación de los siguientes campos:

  • Rol
  • Evento
  • Post
  • Trabajo

Próximamente estará disponible una versión con más funcionalidades.
